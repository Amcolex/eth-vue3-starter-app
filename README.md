# eth-vue3-starter-app
This app lets you quickly start development of Decentralised Apps on Ethereum


## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
